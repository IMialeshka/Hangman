Инструкция:

Запустите приложение; 

Для старта игры введите "да". Для выхода введите "нет". Строчные или заглавные - значения не имеет.


Введите в консоль букву кириллического алфавита (допустимы как строчные, так и заглавные).

Отслеживайте прогресс игры в отображаемом интерфейсе;

По окончанию игры введите в консоль "да"/"нет" для старта еще одного раунда игры или для выхода из игры.
